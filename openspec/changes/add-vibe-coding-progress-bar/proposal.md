# 变更：添加 Vibe Coding 进度栏

## 为什么

AI 辅助编程正在成为主流工作流，但开发者缺乏一个专门的工具来实时监控 AI 代码生成进度。当 AI 执行长时间任务时，开发者只能等待或频繁切换窗口查看状态，这降低了开发效率和用户体验。

## 什么会改变

- 添加一个基于 Tauri 的桌面应用程序作为悬浮窗
- 实现实时进度监控和可视化提醒功能
- 支持快速切换和窗口管理
- 提供与主流 IDE 的集成能力

## 影响

- 受影响的 specs：vibe-coding-progress-bar（新建）
- 受影响的代码：Tauri 桌面应用、前端 UI、进度监控服务
- 外部依赖：Tauri 框架、React/Vue 前端框架
